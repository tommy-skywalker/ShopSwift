# Changelog

All notable changes to ShopSwift will be documented in this file.

## [1.0.0] - 2023-12-10

### Added
- Initial release
- Authentication with JWT
- Product, User, and Order management
- MongoDB integration
- Redis caching
- Email notifications
- Webhook support
- API documentation with Swagger
- Docker support
- CI/CD pipeline
- Testing framework
- Rate limiting
- File upload support
- Search and filtering
- Metrics and monitoring
- Category management

### Security
- JWT authentication
- Password hashing with bcrypt
- Rate limiting
- Helmet security headers
- CORS configuration
