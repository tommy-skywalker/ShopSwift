# ShopSwift

A modern iOS shopping application built with SwiftUI.

## Features

- Browse products
- Shopping cart
- User authentication
- Order management
- Product search and filtering

## Requirements

- iOS 15.0+
- Xcode 14.0+
- Swift 5.7+

## Installation

1. Clone the repository
2. Open `ShopSwift.xcodeproj` in Xcode
3. Build and run
